# Daftar Fitur Frontend yang Bisa Ditambahkan

1. **Order History (User)** _(sedang dikerjakan)_

   - Halaman riwayat pesanan user setelah checkout.

2. **Profile & Pengaturan User** _(sudah dikerjakan)_

   - Halaman profile user sudah tersedia di `/profile`.
   - Fitur: lihat & edit data diri (nama, email, avatar), validasi form (React Hook Form + Zod), dark mode, UI modern, feedback interaktif.
   - Sudah ada unit test untuk validasi form dan tampilan default.

3. **Notifikasi/Feedback UI** _(sudah dikerjakan)_

   - Notifikasi visual (misal: sukses tambah ke cart, checkout, error, dsb) sudah diimplementasikan dengan Sonner Toast.

4. **Validasi Form Lanjutan** _(sudah dikerjakan)_

   - Validasi lebih baik pada form register, signin, dan checkout sudah diimplementasikan (menggunakan React Hook Form + Zod).

5. **Admin Panel (Frontend)**

   - Halaman admin untuk tambah/edit/hapus produk (khusus admin).

6. **Testing Frontend**
   - Penambahan unit test untuk komponen dan halaman utama.

---

Biasanya, fitur order history atau notifikasi feedback jadi prioritas berikutnya untuk meningkatkan pengalaman user. Silakan pilih fitur mana yang ingin didahulukan!
